Autify Genesis

AI-Powered Test Generation

Developed & deployed the initial version of Autify Genesis that generates tests, gherkin & playwright code from PRD specs of a feature.

Show details

Autify Genesis represents a breakthrough in test automation by using LLMs to automatically generate comprehensive test scripts from product requirements. This AI-powered solution significantly reduces the time required for test creation and helps catch edge cases human testers might miss.

The system was built with a custom-trained model fine-tuned specifically for understanding software requirements and generating appropriate test cases. The project involved:

  • Developing a specialized prompting system to extract testing requirements
  • Creating a code generation pipeline for Playwright scripts
  • Building a validation system to ensure generated tests met quality standards
  • Designing an intuitive UI for reviewing and editing generated tests

The project has been successfully deployed in production, allowing teams to generate test suites in minutes rather than days.

Token Compression

Optimizing LLM Performance

This project explores how to compress multiple LLM tokens into a single embedding vector to reduce attention computation.

Show details

As language models grow in size and capability, the quadratic computational cost of attention mechanisms becomes a significant bottleneck. This research project investigated novel approaches to token compression to mitigate this issue.

Key contributions included:

  • Development of a novel token compression algorithm that preserves semantic meaning
  • Benchmarking of compression techniques across various model sizes
  • Analysis of performance impacts on different types of language tasks
  • Open-source implementation for the research community

The results demonstrated a 30% reduction in computational requirements with minimal impact on model performance across standard NLP benchmarks.

In-House OCR System

Japanese & English Character Recognition

Developed & trained an in-house OCR model from scratch using transformers with the character recognition rate of > 95% for 30k+ Japanese and English characters.

Show details

This project addressed the challenge of accurately recognizing both Japanese and English text in screenshots and images, which is crucial for automated testing systems. Existing OCR solutions struggled with mixed-language content and UI elements, necessitating a custom solution.

The development process involved:

  • Building a custom dataset with over 100,000 annotated UI elements
  • Designing a transformer-based architecture optimized for character recognition
  • Implementing specialized data augmentation techniques for UI screenshots
  • Creating an efficient inference pipeline for production use

The resulting system achieved >95% accuracy on Japanese and English characters, significantly outperforming commercial alternatives in this specific domain. The OCR system has been integrated into the core product and processes millions of image segments monthly.

Prompt Engineering Research

Early Applications of LLMs

Pioneered applications of prompt engineering with GPT-2 and T5 models before the technique became mainstream in AI.

Show details

This forward-looking project explored the capabilities of early large language models like GPT-2 and Google's T5 for solving practical problems through clever prompt design. This work predated the later explosion of interest in prompt engineering by several years.

The research included:

  • Systematic evaluation of prompt structures for different task types
  • Development of techniques to handle context limitations
  • Exploration of few-shot learning through examples
  • Creation of a published guide for developers on effective prompting

This work was published as an article on Toptal's engineering blog and has received significant attention as prompt engineering became a central technique in modern AI applications.
